## nvlite 

[nvchad](https://nvchad.com/), but with fewer plugins

```
- nvim-autopairs
- LuaSnip
```

### Installation

Install base and load

```
git clone git@github.com:spikedoanz/nvlite.git ~/.config/nvim && nvim
```

Overwrite configs

```
cp -r extras/configs/ ~/.local/share/nvim/lazy/NvChad/lua/nvchad/
```
